# Ticket QR Generator Worker

A full-stack MERN application that enables organizations to create, manage, and validate digital tickets using QR code technology.

The system replaces manual ticket handling with a secure digital workflow where users can generate tickets, create unique QR codes, scan tickets, and track ticket usage through analytics.

---

# Project Overview

The Ticket QR Generator Worker is designed for event and operational environments where physical tickets or spreadsheet-based tracking create problems such as:

- Manual verification delays
- Duplicate ticket usage
- Poor ticket tracking
- Lack of real-time status visibility

This application provides a centralized digital ticket management platform with QR-based validation.

# Features

## Authentication

- User registration
- User login
- JWT-based authentication
- Secure password hashing
- Protected API routes


## Ticket Management

- Create new tickets
- Generate unique ticket numbers
- Store ticket information
- View all tickets
- View ticket details
- Track ticket status


## QR Code System

- Automatic QR code generation
- Unique QR for every ticket
- QR scanning functionality
- Ticket validation after scanning


## Ticket Status

Supports ticket lifecycle management:

- Active
- Used
- Cancelled

## User Interface

- Professional SaaS dashboard design
- Responsive layout
- Clean authentication pages

- Mobile-friendly interface
